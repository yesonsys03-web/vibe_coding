# 1. 낯선 정거장
겨울밤 공기는 차가웠고, 가로등 불빛 아래 내리는 눈은 유난히 느리게 춤을 췄다. 플랫폼 끝자락에 선 남자는 습관처럼 주머니 속에서 작은 회중시계를 매만졌다.

"기차는 언제 오나요?"
나지막한 목소리에 남자는 뒤를 돌아보았다. 푸른 코트를 입은 소녀가 커다란 트렁크 위에 걸터앉아 그를 올려다보고 있었다.

"글쎄요. 원래 잘 오지 않는 역이니까요."
남자는 시선을 다시 허공으로 던지며 대답했다.

# 2. 어긋난 톱니바퀴
덜컹-
자정이 넘어서야 멀리서 기적이 울렸다. 희미한 불빛이 어둠 속에서 천천히 커지며 플랫폼을 비추었다. 하지만 멈춰 선 기차 문이 열리자마자 소녀의 얼굴은 창백하게 굳어버렸다.

"안 돼..."
소녀가 작게 중얼거렸다. 열차 안은 텅 비어 있었고, 오직 바닥에 버려진 빈 깡통 하나만 이리저리 굴러다니며 쓸쓸한 소리를 내고 있었기 때문이다.
