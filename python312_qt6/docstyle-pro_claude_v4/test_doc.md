# DocStyle Pro 테스트 문서

이 문서는 새롭게 개선된 **템플릿 선택 UI**를 테스트하기 위한 문서입니다.

## 1. 개요
우리는 템플릿 라이브러리를 카테고리별로 정렬하고, 각 템플릿의 시각적 디자인을 고도화했습니다.

### 1-1. 주요 변경 사항
- **카테고리화**: Editorial, Business, Tech 등으로 그룹화
- **디자인 개선**: 그림자 효과, 그라데이션, 둥근 모서리 적용

> [!NOTE]
> 이 박스는 템플릿의 `box_bg`와 `box_border` 색상을 시각적으로 보여줍니다.

## 2. 결론
이제 더 쉽고 빠르게 원하는 스타일을 찾을 수 있습니다.
