# DocStyle Web Frontend (MVP)

## Prerequisites

- Backend API running at `http://127.0.0.1:8000`

## Run

```bash
npm install
npm run dev
```

Open:

`http://127.0.0.1:3000`

## API Base URL

Set `NEXT_PUBLIC_API_BASE` in `.env.local` if your backend is not on `127.0.0.1:8000`.

Example:

```bash
NEXT_PUBLIC_API_BASE=http://127.0.0.1:8000
```
